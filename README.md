# 일본 여행일지

여행 일정, 여행일지, 짐목록, 예산을 관리하는 개인용 여행 플래너 웹앱입니다.
모든 데이터는 브라우저 localStorage에 저장되며 별도 백엔드가 필요 없습니다.

## 개발 실행

```bash
cd japan-travel-journal
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

`dist/` 폴더가 정적 파일로 생성되며, GitHub Pages 등 어떤 정적 호스팅에도 배포할 수 있습니다.

## 기능

- **일정**: 날짜별 일정과 활동(시간/장소/메모)을 추가·정렬·삭제
- **일지**: 날짜와 기분 이모지를 곁들인 하루 기록
- **짐목록**: 카테고리별 체크리스트 (기본 필수품 포함)
- **예산**: 항목별 예산 대비 실지출 관리
